# Security policy

Please report credential exposure or exploitable ClaudeX integration defects privately through GitHub's security-advisory interface rather than a public issue.

Do not attach OAuth files, downstream API keys, complete proxy journals, debug logs containing prompts, or proprietary source code to reports. Redact account identifiers and reproduce with synthetic inputs where possible.

Upstream CLIProxyAPI vulnerabilities should also be reported to its maintainers when the issue is not introduced by this repository's packaging or configuration.
